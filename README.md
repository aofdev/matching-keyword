# Matching Keyword

> Keywords from JSON that match in CSV

## Features ✨
- Include words
- Exclude words
- Support operators AND or `+` to find words containing all terms separated

## Quick Start ⚡️

```bash
cargo run
```

## Tests 🧪

```bash
cargo test
```

## Benchmark 🚀

```bash
cargo bench
```
